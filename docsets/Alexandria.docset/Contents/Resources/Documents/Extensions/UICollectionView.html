Cells
  • registerCell(_:reuseIdentifier:)

    Registers a UICollectionViewCell for use in a UICollectionView.

    By default, the class name of the cell is used as the reuse identifier.

    Example: “` class CustomCell: UICollectionViewCell {}

    let collectionView = UICollectionView()

    // registers the CustomCell class with a reuse identifier of CustomCell collectionView.registerCell(CustomCell) ”`
